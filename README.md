# AI Chatbot Project 🤖

This is a simple chatbot built using:
- Flask (Backend)
- HTML, CSS, JS (Frontend)
- OpenRouter API

## How to run

1. Install dependencies
pip install -r requirements.txt

2. Set API key
set API_KEY=your_key   (Windows)

3. Run server
python app.py

4. Open browser
http://127.0.0.1:5000
